## 📖 项目简介

本仓库致力于梳理 **AI 大模型（Large Language Models, LLMs）** 的发展脉络、关键节点、行业事件、产品发布、技术进展与生态变化。

在这个风起云涌的“大模型江湖”中，我们将以客观、中立的视角，汇总来自互联网的公开信息，为研究者、开发者及爱好者提供一份详实的参考资料。

### ⚠️ 重要声明

1.  **来源透明**：本仓库所有资源均来自互联网公开搜索结果与权威报道。
2.  **客观中立**：内容仅做资料汇总与记录，**无任何二次创作、恶意加工、断章取义或主观篡改**。
3.  **开源共享**：本项目完全开源，遵循 [MIT License](./LICENSE)。

## 📅 发展时间线 (Timeline)

> *（待补充：按年份/月份梳理的关键里程碑）*

- **2017**: Transformer 架构提出 (Google)
- **2018**: GPT-1 (OpenAI), BERT (Google)
- **2019**: GPT-2
- **2020**: GPT-3
- **2022**: ChatGPT 发布，AI 爆发元年
- **2023**: GPT-4, Llama, Midjourney v5, 文心一言等百模大战
- **2024**: Sora, GPT-4o, Claude 3.5, Llama 3...
- **2025**: DeepSeek-R1, Grok-3, Qwen 2.5-Max (推理模型爆发)
- **2026**: Gemini 3, GPT-5 (预计/部分发布)...
