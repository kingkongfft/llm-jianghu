# OpenAI 发展史 (OpenAI History)

OpenAI 是当前全球最顶尖的 AI 研究机构之一，从最初的非营利实验室发展为引领生成式 AI 革命的科技巨头。

## 📅 关键里程碑 (Timeline)

### 2015-2018: 非营利初心与早期探索
- **2015年12月**: OpenAI 在旧金山成立，初始定位为**非营利组织 (Non-profit)**。
    - *创始人*: Sam Altman, Elon Musk, Greg Brockman, Ilya Sutskever, Wojciech Zaremba, John Schulman。
    - *愿景*: 确保通用人工智能 (AGI) 造福全人类。
- **2016年**: 发布 **OpenAI Gym**（强化学习工具包）和 **Universe**。
- **2018年**: 
    - 发布 **GPT-1** (Generative Pre-trained Transformer)，开启了 LLM 的路线。
    - **Elon Musk 退出董事会**，理由是与特斯拉 AI 存在潜在利益冲突。

### 2019-2021: 商业化转型与 GPT-3
- **2019年**:
    - **架构重组**: 成立 **OpenAI LP**（“封顶利润”子公司），旨在筹集资金以支持高昂的算力成本。
    - **微软投资**: Microsoft 宣布向 OpenAI 投资 **10 亿美元**，并成为独家云服务提供商。
    - **GPT-2**: 最初仅面向科研开放，后分阶段发布。
- **2020年**: 发布 **GPT-3** (1750亿参数)，展示了惊人的小样本学习 (Few-shot learning) 能力，震惊业界。
- **2021年**: 发布 **DALL-E** (图像生成) 和 **Codex** (代码生成，GitHub Copilot 的基础)。

### 2022-2023: ChatGPT 爆发与 GPT-4
- **2022年11月**: 发布 **ChatGPT** (基于 GPT-3.5)，仅用 5 天用户破百万，引爆全球 AI 热潮。
- **2023年3月**: 发布 **GPT-4**，支持多模态输入，在推理和考试能力上大幅跃升。
- **2023年11月**: **“宫斗”事件**: Sam Altman 突遭董事会解雇，引发员工集体辞职威胁，5 天后 Altman 官复原职，董事会重组。

### 2024-2026: 产品矩阵爆发与企业级扩张
- **2024年**:
    - **Sora**: 发布文生视频模型，效果震撼。
    - **GPT-4o**: 实现了实时语音、视频交互的端到端能力。
    - **o1 系列**: 发布具有慢思考/推理能力 (Chain of Thought) 的模型，擅长数学与编程。
    - **SearchGPT**: 进军 AI 搜索领域。
- **2025年**:
    - **o3 系列**: 全面推送 o3 模型，解决了长思考的延迟问题，进一步提升了推理速度。
    - **GPT-5 / 5.3-Codex**: 搜索结果提及的新一代模型，预计拥有 **400,000** 上下文窗口和 **128,000** 输出能力，在 SWE-bench Pro 等代码基准测试中大幅领先。
    - **组织架构调整**: 完成重组，明确了 OpenAI LP 与 OpenAI Foundation (非营利) 的关系。
- **2026年 (至今)**:
    - 持续优化 **GPT-6** 原型，具备自我进化能力。
    - 进一步强化 AI 安全与对齐 (Alignment) 技术。

## 🚀 核心模型与产品

| 系列 | 代表模型 | 简介 |
| :--- | :--- | :--- |
| **GPT (文本)** | GPT-4o, GPT-5 (预测) | 核心语言模型，4o 侧重多模态交互，5 侧重全能表现。 |
| **Reasoning (推理)** | o1, o3 | 强化学习训练的慢思考模型，擅长复杂逻辑任务。 |
| **DALL-E (图像)** | DALL-E 3 | 文生图模型，已深度集成于 ChatGPT。 |
| **Sora (视频)** | Sora | 文本生成长达 60 秒的高质量视频（模拟物理世界）。 |
| **Whisper (语音)** | Whisper v3 | 开源的通用语音识别模型 (ASR)。 |
