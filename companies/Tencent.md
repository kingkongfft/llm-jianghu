# Tencent (腾讯)

腾讯作为中国互联网巨头，在人工智能领域拥有深厚的技术积累。其大模型品牌 **腾讯混元 (Tencent Hunyuan)** 采用全链路自研技术，已深度整合进微信、腾讯云、腾讯广告等核心业务中。

## 核心模型 (Core Models)

### 混元大模型 (Hunyuan)
腾讯混元采用 MoE (Mixture-of-Experts) 架构，致力于在性能与推理成本之间取得最佳平衡。

- **Hunyuan-Pro / Standard**: 通用大模型，具备强大的中文创作、逻辑推理和任务执行能力。
- **Hunyuan-Large (2024年底/2025年)**: 开源的 MoE 模型，总参数量达到 389B（激活参数 52B），支持 256k 长上下文。在多项基准测试中表现优异，特别是长文档处理能力。
- **Hunyuan-A13B (2025年6月)**: 高效的 MoE 模型（总参数 80B，激活 13B），专为端侧和低延迟场景设计，性能对标 Llama 3 等主流开源模型。
- **Hunyuan-DiT**: 具有文生图能力的 Diffusion Transformer 模型，支持中英文双语输入，画面生成质量高，尤其对中文古诗词意境理解深刻。

## 技术特点
- **MoE 架构**: 腾讯是 MoE 架构的坚定拥护者，通过稀疏激活技术显著降低了推理成本。
- **全链路自研**: 从算法、机器学习框架 (Angel) 到基础设施，腾讯拥有完整的自研 AI 技术栈。
- **多模态融合**: 在视觉生成、视频生成（跟随 Sora 热潮发布的视频模型）方面也有布局。

## 产品与生态
- **腾讯元宝**: 面向 C 端用户的 AI 助手 App，深度打通微信生态（如公众号文章分析）。
- **腾讯云 MaaS**: 提供混元模型的 API 服务，支持企业级精调和私有化部署。
- **微信集成**: 混元大模型已接入微信读书、腾讯会议、腾讯文档等国民级应用，拥有极大的落地场景优势。

## 关键事件
- **2023年9月**: 腾讯混元大模型正式亮相。
- **2024年5月**: 发布混元文生图模型 (Hunyuan-DiT) 并开源。
- **2024年底**: 发布 Hunyuan-Large 开源模型，展示了其在 MoE 技术上的实力。
- **2025年6月**: 发布 Hunyuan-A13B，进一步完善模型矩阵。
