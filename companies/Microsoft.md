# Microsoft (微软)

Microsoft 是全球领先的科技巨头，在人工智能领域通过自研（Phi 系列）与投资合作（OpenAI）双管齐下的策略，建立了庞大的 AI 生态系统。

## 核心模型 (Core Models)

### Phi 系列 (Small Language Models - SLMs)
Microsoft 专注于开发高性能的**小语言模型 (SLM)**，旨在移动设备和边缘计算场景中提供媲美大模型的推理能力。

- **Phi-1 / Phi-1.5 / Phi-2**: 早期探索版本，验证了高质量教科书级数据对模型能力的提升。
- **Phi-3 系列** (2024年4月发布): 包括 Mini (3.8B), Small (7B), Medium (14B) 等版本。在常识推理、语言理解等基准测试中表现优异，部分指标超越了同等甚至更大参数规模的模型（如 Llama 3 8B）。
- **Phi-Silica** (2024年5月): 专为 Copilot+ PC 设计的 3.3B 参数模型，集成于 Windows 系统底层 NPU 运行。
- **Phi-4** (2024年12月): 14B 参数。主要特点是使用了大量合成数据进行训练，在数学和复杂推理任务上表现突出。
- **Phi-4-reasoning** (2025年5月): 引入了推理（Reasoning）能力的版本，利用 DeepSeek-R1 生成的数据进行微调，优化了思维链（CoT）能力。

### Turing 系列
Microsoft 内部使用的超大规模语言模型系列（如 Turing-NLG），主要用于 Bing 搜索和其他 Azure 认知服务，较少直接面向公众开源。

## 产品与生态 (Products & Ecosystem)

### Microsoft Copilot
Microsoft 将 AI 能力整合为统一的 Copilot 品牌，贯穿其所有产品线：
- **Copilot for Web**: 基于 GPT-4 的对话式搜索助手（原 Bing Chat）。
- **Copilot for Microsoft 365**: 集成于 Word, Excel, PowerPoint, Outlook 等办公软件中，提供文档生成、数据分析、邮件摘要等功能。
- **GitHub Copilot**: 广受开发者欢迎的代码辅助工具，基于 Codex 模型。
- **Copilot+ PC**: 硬件与 AI 深度结合的全新 PC 类别，支持本地运行 SLM（如 Phi-Silica）。

### Azure AI
提供企业级 AI 服务，独家托管 OpenAI 的模型 API（Azure OpenAI Service），并提供 Phi 系列等开源模型的无服务器推理（Model-as-a-Service）。

## 关键事件
- **2019/2023**: 对 OpenAI 进行数十亿美元的战略投资，获得其技术独家商业化授权。
- **2023年2月**: 发布新版 Bing (The New Bing)，集成 GPT-4，引发搜索引擎 AI 革命。
- **2024年**: 大力推行 "AI PC" 概念，发布 Phi-3 系列，确立在端侧模型领域的领导地位。
- **2025年**: 发布 Phi-4 reasoning 系列，进一步探索小模型的推理极限。
