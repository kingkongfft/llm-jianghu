# Moonshot AI (月之暗面)

月之暗面 (Moonshot AI) 是中国最快达到“独角兽”估值的 AI 初创公司之一，由前清华大学计算机系学生杨植麟创立。公司以其旗舰产品 Kimi 智能助手及其背后的超长文本处理能力（Long Context）而闻名。

## 核心模型 (Core Models)

### Kimi 系列
Moonshot AI 的核心竞争力在于长文本处理和多模态理解。

- **Kimi (2023年10月)**: 首发版本即支持 20 万字（200k tokens）上下文输入，成为当时市场上最长的长文本模型。
- **Kimi K2 (2025年7月)**: 拥有万亿级 (1T) 参数的旗舰模型，对标 GPT-5 和 Gemini 3 Pro，宣称在多项基准测试中达到 Claude 级别。
- **Kimi K2.5 (2025年12月/2026年)**: 开源了万亿参数级模型 API，进一步优化了推理（Reasoning）和编码能力，成本显著低于竞品。
  - **特点**: 支持 **256k** 上下文，具备极强的视觉理解（Visual Coding）和 Agent Swarm（智能体集群）能力。
- **Thinking Mode**: 提供了具备深度思考能力的版本 `kimi-k2-thinking-turbo`，能够处理复杂逻辑和长链条推理。

### 技术特点
- **超长上下文 (Long Context)**: Kimi 是最早也是最坚定推行长文本策略的公司，其能够无损处理数百万字的小说、财报或法律文档。
- **视觉编码 (Visual Coding)**: K2.5 模型在理解屏幕截图并将其转换为代码方面表现卓越。
- **Agent Swarm**: 支持多智能体协同工作，能够将复杂任务拆解为子任务并并行处理。

## 产品与生态
- **Kimi 智能助手**: 面向 C 端用户的对话应用，拥有网页版和 App，在中文互联网用户中渗透率极高。
- **Moonshot Open Platform**: 开放平台，提供 API 接入服务，支持开发者构建基于 Kimi 的应用。

## 关键事件
- **2023年10月**: 公司成立仅数月即发布首个千亿参数模型，并宣布支持 20 万字上下文。
- **2024年3月**: 宣布将上下文窗口提升至 200 万字，引发业界轰动。
- **2024年8月**: 活跃用户数跃居国内大模型应用前三。
- **2025年7月**: 发布 Kimi K2 万亿参数模型，正式进入顶级模型竞争行列。
- **2026年**: 持续发布 K2.5 系列，并在开源和 API 价格战中保持极强的竞争力。
