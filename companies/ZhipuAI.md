# Zhipu AI (智谱 AI)

智谱 AI (Zhipu AI) 是源自清华大学计算机系技术成果转化的 AI 初创公司，致力于打造新一代认知智能大模型。作为中国最早开源双语大模型的厂商之一，智谱 AI 在开源社区和企业级市场均拥有极高的影响力。

## 核心模型 (Core Models)

### GLM 系列 (General Language Model)
智谱 AI 的核心产品线，涵盖对话、多模态、代码生成等多个领域。

- **ChatGLM-6B / 3 (2023)**: 早期开源的轻量级模型，以其在消费级显卡上的低成本部署能力引爆了中文开源社区。
- **GLM-4 (2024年1月)**: 全面升级的基座模型，性能比肩 GPT-4。具备强大的多模态理解能力（GLM-4V）和长文本处理能力（128k/1M context）。
  - **GLM-4 All Tools**: 专门优化了工具调用（Function Calling）和 Agent 能力，能够自主调度网页浏览、代码解释器等工具。
- **GLM-4.5 (2025年8月)**: 新一代旗舰模型，拥有 **355B** 参数。
  - **特点**: 采用 MoE（混合专家）架构，专为 Agent 任务设计，具备极强的推理（Reasoning）和代码能力。
  - **GLM-4.5-Air**: 106B 参数的高效版本，平衡了性能与推理成本。
  - **GLM-4.5V**: 多模态版本，支持视觉理解与推理。

### 其他模型
- **CodeGeeX**: 专为代码生成设计的模型，广泛集成于开发者的 IDE 插件中。
- **CogView**: 文本生成图像模型，支持高质量的中文提示词生图。

## 技术特点
- **Agentic AI**: GLM-4.5 系列特别强化了 Agent 能力，支持复杂的任务规划和多步推理，被设计为构建智能体的核心基座。
- **双语优化**: 在中英双语环境下表现均衡且优异，特别是在中文文化理解和长文本摘要方面具有优势。
- **混合推理模式**: 支持 "Thinking Mode"（深度思考）和快速模式，适应不同场景需求。

## 产品与生态
- **智谱清言 (ChatGLM App)**: 面向 C 端用户的智能助手应用，集成了对话、数据分析、绘画等功能。
- **MaaS 开放平台 (bigmodel.cn)**: 提供全系模型的 API 服务，支持企业进行微调和私有化部署。

## 关键事件
- **2023年**: 开源 ChatGLM-6B，迅速成为中文社区最受欢迎的开源模型之一。
- **2024年1月**: 发布 GLM-4，宣布在综合能力上逼近 GPT-4。
- **2025年8月**: 发布 GLM-4.5 系列，进一步确立在 Agent 和推理能力上的领先地位，模型参数量达到 355B。
