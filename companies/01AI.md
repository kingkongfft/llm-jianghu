# 01.AI (零一万物)

01.AI 由全球知名人工智能专家李开复博士创办，以“Global First”的理念打造，致力于研发最先进的开源大模型。其旗舰系列 **Yi** 在全球开源社区中备受赞誉，以其卓越的性价比和技术前瞻性（TC-PMF）闻名。

## 核心模型 (Core Models)

### Yi 系列 (Yi Series)
Yi 系列是 01.AI 的主力产品，兼具开源与闭源版本。

- **Yi-34B (2023年11月)**: 开源双语大模型，是当时全球最强的 34B 规模模型，超越 Llama 2 70B。支持 200k 超长上下文，不仅中文能力强，在英文评测榜单上也名列前茅。
- **Yi-Large (2024年5月)**: 闭源旗舰模型，参数规模在千亿级。其在 reasoning (推理) 和 math (数学) 上的表现媲美 GPT-4 Turbo。
- **Yi-XLarge (2024/2025年)**: 下一代 MoE (Mixture-of-Experts) 模型，迈向万亿参数规模，旨在进一步挑战 AGI 边界。
- **Yi-Vision**: 多模态视觉理解模型，支持高分辨率图像输入。

### TC-PMF (Technology-Cost Product Market Fit)
01.AI 提出的核心理念，强调不仅要在技术上领先，更要在推理成本（Cost）上做到极致优化。Yi 系列模型在相同性能下，往往具有更低的显存占用和更快的推理速度。

## 产品与生态
- **万知 (Wanzhi)**: 面向 C 端用户的生产力助手，集成了 Yi-Large 的能力，专注于长文档分析、PPT 生成和会议纪要。
- **Yi API**: 提供高性能、低延迟的模型 API 服务，不仅支持 OpenAI 兼容接口，还提供 Function Calling 等高级特性。
- **开源贡献**: Yi-34B 的开源极大地丰富了开源社区的选择，被众多开发者微调用于各种垂直领域。

## 关键事件
- **2023年11月**: 发布 Yi-34B，迅速登顶 Hugging Face Open LLM Leaderboard，引发全球关注。
- **2024年5月**: 发布 Yi-Large 闭源模型，在 LMSYS Chatbot Arena 中取得优异排名。
- **2024年**: 完成新一轮融资，估值超过 10 亿美元，确立其独角兽地位。
- **2025年**: 持续迭代 Yi-Large 和 Yi-XLarge，并在多模态领域发力。
