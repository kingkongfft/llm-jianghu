# Models

本目录收录各模型详细参数、发布时间、开源情况、Benchmark 表现等。

## 收录标准
- 有公开论文或技术报告
- 有明确的发布日期
- 在社区有一定讨论度

## 目录结构
可以按模型名/系列名创建文件，例如 `GPT.md`, `Llama.md`, `Qwen.md`。
