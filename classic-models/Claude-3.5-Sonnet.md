# Claude 3.5 Sonnet

**Claude 3.5 Sonnet** 是 Anthropic 于 2024 年 6 月发布的中端高性能模型。它凭借卓越的编程能力和速度迅速成为开发者的最爱，性能往往超越上一代旗舰 Claude 3 Opus。

## 技术规格 (Technical Specifications)

- **发布日期**: 2024 年 6 月 20 日
- **开发者**: [Anthropic](../companies/Anthropic.md)
- **架构**: 基于 Transformer
- **上下文窗口**: 200,000 tokens
- **输出限制**: 8,192 tokens
- **知识截止日期**: 2024 年 4 月

## 核心特性 (Key Features)

- **编程精通**: 发布时被公认为最佳编程模型，解决了 64% 的内部 Agent 编程评估问题。
- **Artifacts**: 随 "Artifacts" UI 功能一同推出，允许模型在专用窗口中生成和渲染代码、文档及图表。
- **速度**: 运行速度是 Claude 3 Opus 的两倍，且价格显著更低。
- **视觉能力**: 发布时在 Claude 3 家族中拥有最强的视觉能力。

## 基准测试 (Benchmarks)

| 基准测试 | 得分 | 备注 |
| :--- | :--- | :--- |
| **MMLU** | 88.7% | 通用知识 (5-shot) |
| **GPQA** | 59.4% | 研究生级推理 (Diamond) |
| **HumanEval** | 92.0% | 代码能力 |
| **MATH** | 71.1% | 数学推理 |

## 影响 (Impact)

Claude 3.5 Sonnet 改变了行业对“中端”模型的认知，证明了更小、更快的模型可以超越仅仅几个月前的更大“前沿”模型。其与 "Artifacts" UI 的结合开创了与大语言模型交互的新方式，从简单的聊天转向协作创作。
