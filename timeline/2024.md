# 2024年：应用落地与推理提升 (The Year of Application & Reasoning)

2024年，大模型行业从单纯的参数竞赛转向应用落地、多模态融合以及推理能力的深度优化。开源模型在这一年取得了里程碑式的胜利，性能上开始全面追平甚至超越顶尖闭源模型。

## 2月 (February)
- **Google**: 发布 **Gemini 1.5 Pro**。
  - **特点**: 引入了突破性的 **100万 (1M)** token 上下文窗口（后续扩展至 2M），能够一次性处理海量视频、音频和代码库，彻底改变了长文本处理的标准。
- **OpenAI**: 发布 **Sora** (文生视频模型)。
  - **特点**: 能够生成长达 60 秒的高清视频，画面一致性、物理规律遵循度极其惊人，引发了全球对视频生成技术的狂热关注。
- **Mistral AI**: 发布 **Mistral Large**，性能直逼 GPT-4，并宣布与微软建立深度合作关系。

## 3月 (March)
- **Anthropic**: 发布 **Claude 3** 系列 (Haiku, Sonnet, Opus)。
  - **特点**: Claude 3 Opus 在 LMSYS Chatbot Arena 等权威榜单上首次超越 GPT-4，终结了 OpenAI 长达一年的霸榜地位。Sonnet 版本以极高的性价比成为开发者首选。
- **Grok**: Elon Musk 宣布开源 **Grok-1** (314B 参数)，成为当时参数量最大的开源模型。
- **Apple**: 发布 MM1 多模态大模型论文，并在随后 WWDC 上推出 **Apple Intelligence**，宣布与 OpenAI 合作。

## 4月 (April)
- **Meta**: 发布 **Llama 3** (8B, 70B)。
  - **特点**: 在同等参数规模下性能碾压对手，70B 版本表现优异，重新定义了开源模型的标准。
- **Microsoft**: 发布 **Phi-3** 系列小模型，展示了高质量数据训练下小模型的惊人潜力。
- **Snowflake**: 发布 **Arctic** 开源模型，专注于企业级任务。

## 5月 (May)
- **OpenAI**: 发布 **GPT-4o** ("o" 代表 omni)。
  - **特点**: 原生多模态（音频、视觉、文本统一端到端），实时语音响应速度达到人类级别（~320ms），且 API 价格减半，速度提升 2 倍。
- **Google I/O**: 发布 **Gemini 1.5 Flash**（轻量级、低延迟）、**Veo**（视频生成）和 **Project Astra**（实时 AI 助手原型）。
- **01.AI**: 发布 **Yi-Large**，在各项评测中表现出色。

## 6月 (June)
- **阿里**: 发布 **Qwen2** 系列。
  - **特点**: Qwen2-72B 在多项基准测试中超越 Llama 3 70B，成为当时全球最强开源模型，且支持 27 种语言。
- **Anthropic**: 发布 **Claude 3.5 Sonnet**。
  - **特点**: 速度是 Opus 的 2 倍，价格是其 1/5，但在编码、推理等多项能力上反超 Opus，设立了新的性价比标杆。推出 **Artifacts** 功能，改变了人机交互形态。

## 7月 (July)
- **Meta**: 正式发布 **Llama 3.1**，包括期待已久的 **405B** 版本。
  - **特点**: Llama 3.1 405B 是首个在性能上真正匹敌 GPT-4o 的开源模型，不仅开源权重，还鼓励使用其生成的数据蒸馏小模型。
- **Mistral AI**: 发布 **Mistral Large 2** (123B)，在代码和推理能力上表现卓越，且显存占用更小。

## 9月 (September)
- **OpenAI**: 发布 **o1** 系列 (原草莓项目 Q*)。
  - **特点**: 引入了 "System 2" 思维模式，通过强化学习进行长链条思维（Chain of Thought），在数学、物理、编程竞赛（IOI/IMO）中达到人类金牌水平。
- **Mistral AI**: 发布 **Pixtral 12B**，首个具备视觉能力的开源多模态模型。

## 10月 (October)
- **Anthropic**: 发布 **Claude 3.5 Haiku** 和升级版 **Claude 3.5 Sonnet**（引入 Computer Use 能力，可操控电脑）。

## 11月 (November)
- **Google**: 发布 **Gemini 2.0** 预览版，进一步提升多模态理解和生成能力。

## 12月 (December)
- **DeepSeek**: 发布 **DeepSeek-V3**。
  - **特点**: 采用创新的 MLA (Multi-Head Latent Attention) 和 DeepSeekMoE 架构，以极低的训练成本（不到 600 万美元）实现了顶尖性能，API 价格极低，引发全球震动。
