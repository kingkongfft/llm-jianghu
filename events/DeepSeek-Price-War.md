# DeepSeek 价格战 (DeepSeek Price War)

**时间**: 2024年末 - 2025年初  
**核心厂商**: DeepSeek (深度求索), OpenAI, Microsoft, Google, AWS  
**关键事件**: DeepSeek V3/R1 发布及 API 降价

## 背景
在 2024 年之前，大模型的推理成本（Inference Cost）一直是制约 AI 大规模应用的主要瓶颈。OpenAI 的 GPT-4 等前沿模型价格高昂，导致开发者和企业难以构建高频调用的应用。

## 经过
### 1. 第一枪：DeepSeek V3 发布
2024 年底，DeepSeek 发布了 V3 模型，并给出了极具破坏力的定价：
- **输入**: $0.14 / 1M tokens
- **输出**: $0.28 / 1M tokens
这一价格仅为当时 GPT-4o 的几十分之一，且性能在多数基准测试中与之持平。

### 2. 高潮：DeepSeek R1 与“1月震荡”
2025 年 1 月，DeepSeek 发布推理模型 **DeepSeek-R1**。
- **性能**: 在数学和代码任务上匹敌 OpenAI o1。
- **价格**: 保持了极低的定价策略 ($0.55 / 1M Input, $2.19 / 1M Output)。
- **市场反应**: 
  - **股市**: 1月27日，受 DeepSeek“低成本高性能”技术路线的冲击，Nvidia 及美国科技巨头股价大幅下跌，市场开始重新评估 AI 算力需求的增长逻辑。
  - **竞品跟进**: 随后，OpenAI、Google 和 Anthropic 等巨头被迫跟进，纷纷推出“Mini”版模型或直接下调 API 价格。

## 影响
1.  **推理成本白菜价**: AI 应用的边际成本趋近于零，催生了大量 Agent 和长思维链应用的落地。
2.  **算力去魅**: 证明了通过算法优化（如 MoE 架构、FP8 训练）可以大幅减少对高端 GPU 的依赖，打破了“大力出奇迹”的算力垄断神话。
3.  **开源崛起**: DeepSeek R1 的开源策略迫使闭源厂商必须提供明显更强的能力才能维持溢价，否则将失去开发者生态。
