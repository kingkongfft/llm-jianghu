# Key Events

本目录收录 AI 行业的重大事件、争议、政策变化与产品发布会。

## 收录标准
- 对行业产生深远影响
- 广泛讨论的热点事件
- 重要政策法规的颁布

## 目录结构
建议按事件类型或主题创建文件，例如 `Policy.md` (政策), `Controversies.md` (争议), `Conferences.md` (会议/发布会)。

## 示例 (Policy.md)

### 政策法规 (Policy & Regulation)

- **2023.07**: 中国网信办发布《生成式人工智能服务管理暂行办法》，8月15日起施行。
- **2024.03**: 欧盟议会正式通过《人工智能法案》(AI Act)，成为全球首部全面监管 AI 的法律。
