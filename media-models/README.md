# 🎨 多模态与媒体模型 (Media Models)

本目录收录了专注于图像、视频、音频生成及多模态理解的前沿 AI 模型。与传统的文本大语言模型（LLM）不同，这些模型主要输出非文本内容，是 AI 创意产业的核心引擎。

## 📹 视频生成 (Video Generation)

| 模型名称 | 开发商 | 发布时间 | 特点 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **[Sora](Sora.md)** | OpenAI | 2024.02 | 物理世界模拟，长达 60s 高清视频 | 开启了 AI 视频元年 |
| **Gen-3 Alpha** | Runway | 2024.06 | 极高的可控性和逼真度 | 影视行业首选工具之一 |
| **Kling (可灵)** | 快手 (Kuaishou) | 2024.06 | 生成 1080p 视频，并在中国市场率先公测 | 效果惊艳，迅速普及 |
| **Vidu** | 生数科技 (Shengshu) | 2024.04 | 单次生成 16s 高清视频 | 清华系团队出品 |
