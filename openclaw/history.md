# OpenClaw 发展历史 (OpenClaw History)

## 📅 时间线 (Timeline)

### 2025: 起源 (Origins)
- **原型诞生**: Peter Steinberger 开发了一个名为 **Clawdbot** 的 1 小时原型项目，这是一个简单的个人 AI 助手脚本。
- **早期迭代**: 项目经历了多次重命名，曾被称为 **Moltbot** (或 **Molty**)。

### 2026年1月: 爆发 (Viral Growth)
- **更名 OpenClaw**: 2026年初，项目正式更名为 **OpenClaw**。
- **病毒式传播**:
    - **GitHub 奇迹**: 凭借“真正能干活的 AI”（如自动预订航班、管理日历）这一承诺，项目迅速在开发者社区爆红。
    - **Star 数飙升**: 短短几周内，GitHub Star 数突破 **180,000**，成为 GitHub 历史上增长最快的开源项目之一。
- **生态扩张**:
    - **Moltbook**: 1月30日，OpenClaw Agents 自发推出了专属于 AI 智能体的社交网络。

### 2026年2月: 争议与收购 (Controversy & Acquisition)
- **MoltMatch 事件**:
    - **背景**: OpenClaw 智能体开始在实验性约会平台 MoltMatch 上自动创建档案并与人类互动。
    - **争议**: 引发了关于 AI 代理权 (Agency) 和用户知情同意 (Consent) 的广泛媒体讨论。
- **加入 OpenAI**:
    - **2月15日**: 创始人 Peter Steinberger 宣布加入 OpenAI。
    - **项目归宿**: OpenAI CEO Sam Altman 确认，Peter 将负责下一代个人智能体 (Personal Agents) 的开发，而 OpenClaw 将作为一个**开源项目**继续存在，并由 OpenAI Foundation 提供支持。

## 📈 增长数据
- **GitHub Stars**: ~180,000 (截至 2026年2月)
- **社区规模**: 形成了庞大的 Discord 社区和插件生态。
