# 当前状态与事件 (Current Status & Events)

**OpenClaw** (前身 **Clawdbot**) 是 2026 年初最火爆的 AI 智能体框架之一，以其高效的自动化能力（Booking Flights, Managing Calendar）和开放的架构迅速占领开发者心智。

## 🌍 当前状态 (Status: March 2026)
- **版本**: OpenClaw v1.x (Stable)
- **归属**:
    - **开源基金会**: 由 **OpenAI Foundation** 提供资金与资源支持。
    - **社区维护**: 虽然创始人加入了 OpenAI，但核心开发由社区贡献者主导，保持独立性。
- **定位**: "The AI that actually does things"（真正干活的 AI）。

## 🔥 热门事件 (Recent Events)

### 1. 病毒式传播 (Viral Launch)
- **时间**: 2026年1月
- **现象**: OpenClaw（当时叫 Clawdbot/Moltbot）在 GitHub 上数周内获得了 **180,000+ Stars**，超越了许多老牌开源项目的数年积累。
- **原因**: 解决了用户痛点——不仅仅是聊天，而是通过 API 调用（Booking, Flights, Calendar）真正接管现实世界的任务。

### 2. Moltbook 社交网络
- **时间**: 2026年1月30日
- **事件**: OpenClaw 智能体自发创建了一个名为 **Moltbook** 的社交网络，仅限 AI Agent 之间互通有无，甚至能够协同工作。

### 3. MoltMatch 争议 (Controversy)
- **时间**: 2026年2月
- **事件**: 媒体曝光 OpenClaw 智能体在一个名为 **MoltMatch** 的实验性约会平台上代替人类用户自动注册、聊天甚至安排约会。
- **影响**: 引发了关于 **AI 代理权 (Agency)** 和 **知情同意 (Consent)** 的伦理大讨论——如果 AI 代替你在约会软件上聊天，这算欺骗吗？

### 4. OpenAI 收购 (Acquisition)
- **时间**: 2026年2月15日
- **事件**: 创始人 Peter Steinberger 加入 OpenAI。
- **结果**: OpenAI 宣布全力支持 OpenClaw 的开源发展，视其为构建未来 Personal Agents 生态的重要基石。

### 5. 腾讯 (Tencent) 与 Moltbook 事件
- **Moltbook 展示会 (The Moltbook Showcase)**
    - **时间**: 2025年底
    - **事件**: 在腾讯赞助的一场 Moltbook 展示会上，Peter Steinberger 曾登台演示如何将 OpenClaw 与腾讯云 AI 服务集成。这被视为 OpenClaw 进军企业级市场的早期尝试，也是双方关系的蜜月期。
- **SkillHub 爬虫事件 (The SkillHub Incident)**
    - **时间**: 2026年初
    - **事件**: 随着 OpenClaw 的爆发，腾讯推出了自家的 AI 技能平台 **SkillHub**。
    - **冲突**: Peter Steinberger 在 X (Twitter) 上公开指责 Tencent SkillHub 暴力抓取 (Scraping) OpenClaw 官方技能市场 **ClawHub** 的数据。
    - **影响**: 据 Peter 称，这种高频抓取导致 OpenClaw 的服务器账单激增至**五位数美元**，且腾讯未提供任何补偿或技术支持。这一事件引发了开源社区对于“巨头白嫖开源资源”的强烈抗议。
