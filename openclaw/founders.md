# 创始人背景 (Founder: Peter Steinberger)

**Peter Steinberger**，奥地利/全球知名的开发者，OpenClaw (前身为 Moltbot, Clawdbot) 的创建者。

## 📅 生平与职业
- **早期经历**:
    - **PSPDFKit 创始人**: 在 OpenClaw 之前，Peter Steinberger 是 **PSPDFKit** 的创始人，这是一家知名的 PDF 开发工具公司，其 SDK 被广泛应用于全球各地的 iOS 和 Android 应用中。
    - **独立开发者**: 长期活跃于开发者社区，以高效率、简洁的代码风格著称。

## 🤖 与 AI 的渊源
- **2025年**: 创建 **Clawdbot**，最初只是一个 1 小时的个人 AI 助手原型，旨在自动化处理琐碎任务（如日历、预订）。
- **2026年**: 随着项目的病毒式传播，将其更名为 **OpenClaw**，并迅速成为 GitHub 上最受欢迎的开源 AI 项目之一。

## 💼 当前职务
- **2026年2月 - 至今**: 加入 **OpenAI**，担任 **下一代个人智能体 (Personal Agents)** 研发负责人。
- **角色定位**: Sam Altman 称其将推动 OpenAI 在 AI Agent 领域的战略落地，而 OpenClaw 则转为由基金会支持的独立开源项目。

## 🗣️ 评价
- **技术极简主义者**: Peter 的开发风格强调实用主义，“AI that actually does things”（真正做事的 AI）是 OpenClaw 能够迅速获得社区共鸣的关键。
- **开源精神**: 即便在被 OpenAI 吸纳后，依然坚持 OpenClaw 的开源属性。
