# 创始人背景 (Founder: Peter Steinberger)

**Peter Steinberger**，奥地利/全球知名的开发者，OpenClaw (前身为 Moltbot, Clawdbot) 的创建者。

## 📅 生平与职业
- **早期经历**:
    - **PSPDFKit 创始人 (2011-2025)**: Peter Steinberger 于 2011 年创办了 **PSPDFKit**（后更名为 Nutrient），这是一家知名的 PDF 开发工具公司。
        - **成就**: PSPDFKit 的 SDK 被 Dropbox, Autodesk, IBM 等全球数千家企业使用，成为 iOS/Android 开发领域的标杆级产品。
        - **风格**: Peter 以极度关注细节（Pixel Perfect）和高质量的代码工程能力著称，在 Apple 开发者社区拥有极高声望。
    - **独立开发者**: 长期活跃于开发者社区，经常分享关于 iOS 开发、逆向工程和高性能计算的深度文章。

## 🤖 与 AI 的渊源 (The OpenClaw Story)
- **缘起 (2025年11月)**: 
    - **周末黑客松**: 在一个周五的晚上，Peter 为了解决自己生活中琐碎的任务（如自动回复邮件、整理日历），用一个周末的时间写出了 **Clawdbot** 的原型。
    - **设计哲学**: 与当时充斥市场的“聊天机器人”不同，Clawdbot 从第一行代码开始就是为了**执行动作 (Action-Oriented)** 而生。
    - **病毒式传播 (2026年1月)**: 
        - 项目在 GitHub 开源后，因其“真正能干活”的特性迅速引爆社区。
        - **数据**: 短短几周内获得 **180,000+ Stars**，每周有超过 **200万** 独立访客访问项目主页，被 Lex Fridman 称为“GitHub 历史上增长最快的项目”。
        - **趣闻**: Peter 在采访中提到，他最初只是想做一个“好玩的玩具”，没想到会演变成一场 AI Agent 的革命。

## 💼 当前职务
- **2026年2月 - 至今**: 加入 **OpenAI**，担任 **下一代个人智能体 (Personal Agents)** 研发负责人。
- **角色定位**: Sam Altman 称其将推动 OpenAI 在 AI Agent 领域的战略落地。Peter 本人则表示，加入 OpenAI 是为了让 OpenClaw 的愿景（每个人都有自己的 Jarvis）能够依托更强大的基础模型得以实现。
- **开源承诺**: 尽管加入 OpenAI，OpenClaw 仍保留为独立开源项目，由 OpenAI Foundation 提供长期支持。

## 🗣️ 评价
- **技术极简主义者**: "Don't build features, build capabilities."（不要构建功能，要构建能力。）
- **社区领袖**: 他证明了在 AI 时代，一个优秀的工程师凭借敏锐的产品直觉，依然可以单枪匹马挑战巨头，并最终赢得尊重。
